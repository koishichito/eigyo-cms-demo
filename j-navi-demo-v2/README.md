# J-Navi デモ（Sales Partner / セールスパートナー）

PRD（MVP）＋追加要望（広告主・広告枠・商談フロー・ネットワーク配分の可視化）に寄せて作成した **フロントエンドのみのデモ** です。  
※ データはサーバーへ送信されず、ブラウザの `localStorage` に保存されます。

---

## できること（このデモの範囲）

### ✅ 基本（セールスパートナー向け）
- サイドバー（PC）/ ハンバーガー（モバイル）でページ遷移
- **ホーム**：KPI（チーム売上/未確定報酬/出金可能額/ランク）・売上トレンド・最近のアクティビティ・お知らせ
- **案件を探す（商材）**：
  - 商品一覧（サイネージ/LED/AI動画）
  - 詳細（仕様/ターゲット/資料DL）
  - 紹介リンク発行（コピー＆リンクを開く）
  - 「想定の報酬内訳（参考）」のプレビュー表示
- **商談管理**：
  - 受信したリード一覧（ステータス管理）
  - リード詳細（紹介チェーン：コネクター/上位/担当代理店）
  - 成約登録（売上・報酬率の入力 → 報酬内訳プレビュー → 取引作成）
- **報酬管理**：
  - 取引履歴（売上/報酬原資/あなたの報酬/ステータス）
  - 内訳モーダルで「代理店/コネクター/上位/J-Navi」の配分%と金額を確認
  - 出金申請（デモ）
- **パートナー管理**：
  - 組織図（紹介関係のツリー）
  - チーム一覧（関与売上/獲得報酬/紹介コード）
  - 招待リンクのコピー＆「招待シミュレート（デモ）」
- **設定**：
  - 表示ユーザー切替（代理店視点⇄コネクター視点）
  - デモデータ初期化

### ✅ 広告領域（今後の拡張に寄せた部分）
- **広告管理**：
  - 広告枠（サイネージ枠）一覧
  - 広告主募集リンク発行（リンク先で「出稿問い合わせ」→ リード作成）
  - 広告主一覧（追加フォーム付き：デモ）
  - 広告リード一覧（問い合わせの蓄積）

## 使い方（ローカル起動）

> Node.js が必要です（Vite 7 系：Node 20.19+ 推奨）

```bash
npm install
npm run dev
```

起動後、表示される Local の URL（例: http://localhost:5173）を開いてください。  
このデモは **HashRouter** なので、URL は `/#/dashboard` のような形式になります。

---

## デモの流れ（おすすめ）

1. **案件を探す** → 任意の商品で「リンク発行」→「リンクを開く」
2. 開いたページ（紹介リンク）でフォーム送信  
   → **商談管理** にリードが作成されます
3. **商談管理** → リード詳細 → 「成約登録」  
   → **報酬管理** に取引が作成され、内訳が見られます
4. **設定** → 表示ユーザー切替  
   → コネクター/代理店で配分の見え方を確認できます

## ディレクトリ

- `src/state/*` … ローカルDB（Context）/ 型 / セレクタ / アクション
- `src/pages/*` … 画面（ホーム/案件/商談/広告/ネットワーク/報酬/設定/紹介LP/招待）
- `public/materials/*` … 資料ダウンロード用のサンプル PDF（デモ）
